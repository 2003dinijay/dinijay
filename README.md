# dinijay

> CLI business card for **Dinithi Nanayakkara** — Full-Stack Developer & UX/UI Designer

## Usage

Run instantly with npx — no install needed:

```bash
npx dinijay
```

## Use as a Library

```bash
npm install dinijay
```

```js
import { run, data } from 'dinijay';

// Run the full CLI card
run();

// Or access the raw data
console.log(data.name);      // "Dinithi Nanayakkara"
console.log(data.skills);    // { Frontend: "React, Next.js, ..." ... }
console.log(data.projects);  // [ { name, desc }, ... ]
console.log(data.links);     // { github, linkedin, email, portfolio }
```
